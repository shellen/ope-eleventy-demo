How This Works

This page uses two open APIs, entirely client-side:

  • Bluesky Public API (public.api.bsky.app) — profile and posts, no auth required
  • Constellation by Microcosm (constellation.microcosm.blue) — network-wide interaction counts from the AT Protocol firehose

The did:plc identity from AT Protocol maps directly to an OPE publisherId. No new identity system needed — any Bluesky user already has a decentralized identifier that works with OPE's portable entitlement model.
